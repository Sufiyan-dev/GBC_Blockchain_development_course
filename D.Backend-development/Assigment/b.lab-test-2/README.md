## Running Steps
1. Install dependencies
    - `/Frontend`
        - ```bash
            npm install
          ```
    - `/Backend`
        - ```bash
            npm install
          ```
2. Running 
    - `/Frontend`
        - ```bash
            npm start
          ```
    - `/Backend`
        - Running backend
        ```bash
            npm run dev
        ```

        - Running local node
        ```bash
            npx hardhat node
        ```
